		PÁGINA DE CADASTRO	
Para a tabela de plantas e inputs na área de cadastro:
	
	- id;
	- data publicação;
	- nome cientifico;
	- família;
	- gênero;
	- ocorrência;
	- tipo de solo;
	- frequência e rega;
	- tempo de vida;
	- dimorfismo sexual;
	- dificuldade;
	- exposição ao sol;
	
	
	Tabela auxiliar plantas_nomes_populares:
		
		- id planta;
		- nome popular;
